В программе реализованы эффкты:
1)Попиксельный расчет освещенности по стандартным моделям 
2)Двумерный пост-эффекты(обводка куба)
3)Normal mapping
4)Parallax mapping
5)Фон
Для сборки нужно скачать данный репозиторий полностью(git clone). Все необходимые файлы и библиотеки лежат в папках. 
При скачивании, будет автоматически скачан проект mg.sln. Его запуск запускает программу.
